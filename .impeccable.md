# Impeccable design context

## Design Context

### Users
Software developers browsing the **nautilai** marketplace of Claude Code plugins. Context: at their terminal/editor, scanning for a relevant tool, wanting to copy an install command fast. Two modes to serve:
- **First-timers** — may not know the marketplace model; need to grasp *what installing does* and that there's a **two-step** flow (add the marketplace once, then install a plugin).
- **Power users** — want to find the right plugin and copy its command in seconds (search/category/filter matters as the list grows).

### Brand Personality
**Deep · curious · alive.** Wonder-of-the-deep and exploration; the nautilus as a *living, chambered, growing* shell — discovery and growth, not a static logo. Confident but not loud; scientific-adjacent without being clinical. Emotional goal: intrigue + trust — a developer should think "how was this made?", not "which AI made this?"

### Aesthetic Direction
- **Theme: depth gradient (light → deep).** Surface-light at the top of the page descending to deep-dark toward the bottom — *the scroll is a dive*. Not a flat dark theme, not a flat light theme; the gradient carries the metaphor.
- **Palette:** OKLCH, sunlit-shallows → abyssal-deep ocean range; neutrals tinted toward a single marine hue for cohesion. Reduce chroma toward the light/dark extremes.
- **Nautilus nod: confident, not gimmicky.** One memorable logarithmic-spiral / chambered mark and chambered *rhythm* in a few select places; cards themselves stay conventional and scannable.
- **Typography:** a distinctive display face with personality (NOT Inter or the reflex defaults) paired with a refined body face; real hierarchy (≥1.25 step ratio). **Monospace only for actual commands.**
- **Anti-references (do NOT look like):** generic dark-SaaS template, Tailwind slate-900 + neon/cyan/purple, gradient text, dark-mode glows, glassmorphism, icon-tile-above-heading card grids, Inter everywhere, eight identical equal-weight cards.

### Design Principles
1. **The dive is the metaphor.** Depth increases with scroll; make light/dark mean "shallow/deep," don't just decorate.
2. **Confident nod, not gimmick.** One strong spiral mark + chambered rhythm; keep plugin cards scannable — devs are here to find a tool fast.
3. **Earned ornament only.** No gradient text, glows, or glass. Every decorative element must reinforce *deep / curious / alive* or be cut.
4. **Distinctive type, real hierarchy.** Display face with character + refined body; mono reserved for commands; clear size contrast.
5. **Don't regress craft.** Preserve `:focus-visible`, `prefers-reduced-motion`, and the `aria-live` clipboard UX. Stay a single self-contained `index.html` (inline CSS/JS, no build step).
6. **Serve both first-timers and power users.** Orient newcomers (what's a marketplace, the two-step install) and give power users fast findability (category/filter) as the catalog grows.
