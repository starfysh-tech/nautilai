Plugins in the reef

Each plugin is self-contained and installs straight into Claude Code.

commitcraft

AI git workflow toolkit — conventional commits, issue validation, PR creation, branch-protection provisioning, and release guidance. Six workflows, one skill.

commitpushprreleasesetupcheck
Read the docs
commitcraft
# install
/plugin install commitcraft@nautilai
# then run
/commitcraft commit # AI conventional commit
/commitcraft push # commit + push, issue-aware
/commitcraft pr # PR with AI description
/commitcraft release # version bump + notes
/commitcraft setup # configure tooling, in chat
/commitcraft check # validate configuration

handoff

Compact the current conversation into a handoff document so a fresh agent can pick up the work — artifacts referenced by path, not restated. Triggers on /handoff or natural language like "hand this off".

cc-adoption-audit

Audits your current Claude Code setup — config, installed plugins/skills/MCP, and project stack — against available features, then outputs prioritized adoption and setup recommendations. Defers "what's new" to /explain-cc-changes; makes no "remove unused" claims.
